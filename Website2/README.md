# Weather-Based Travel Planner
link: https://webdev-2026-spring-dbsk45zcr-echodong7-6015s-projects.vercel.app/
A modern, responsive web application that helps users plan their travel activities based on real-time weather conditions. Get personalized activity suggestions powered by AI to make the most of your trip!

## 🚀 Technologies Used
### APIs
- **Geocoding API**: Converts city names to coordinates and reverse geocoding
- **Weather API**: Fetches real-time weather data
- **OpenAI API**: Generates personalized activity suggestions
- **Geolocation API**: Browser API for accessing user's current location

## 📱 How to Use

1. **Choose Your Destination**:
   - Enter a city name in the search field (e.g., "New York", "Tokyo", "Paris")
   - Or click "Use Current Location" to get weather for your current position

2. **View Weather Information**:
   - See current temperature (Fahrenheit and Celsius)
   - Check UV index for sun protection planning
   - Review weather conditions and wind speed

3. **Get Activity Suggestions**:
   - AI-powered recommendations based on current weather
   - 5 personalized activities tailored to the conditions
